# Desktop 22 - Parts List

I use this machine for gaming, media consumption and occasional coding.

[PCPartPicker Part List](https://pcpartpicker.com/list/kb9nKp)

**Total Cost: $1250.79**

## CPU

_Intel Core i7-13700K 3.4 GHz 16-Core Processor ($449.99 @ Amazon)_

The i7 was chosen because it should have good performance while still being $300 cheaper than the i9. I am spending just under half the total build price on the CPU which feels right for this kind of build.

## CPU Cooler

_Thermalright Peerless Assassin 66.17 CFM CPU Cooler ($41.30 @ Amazon)_

This CPU Cooler was chosen because it is cheap and Tom's Hardware gave it a really [good review](https://www.tomshardware.com/reviews/thermalright-peerless-assassin-120-se). According to the Amazon reviews, it comes with Thermal paste.

## Motherboard

_Asus PRIME Z690-P D4 ATX LGA1700 Motherboard ($199.99 @ Amazon)_

This motherboard was chosen because it supports DDR4. It is also very similar to the motherboard I used in my last build. It also has headers for a front-panel USB-C.

This motherboard has Z690 chipset. This chipset was released with the 12th Gen Intel CPUs. It will work with 13th Gen if the BIOS is upgraded to >= v1620. After assembly, I took the machine to Microcenter and had them perform the BIOS update. They put in a 12th Gen, installed the BIOS update, then put my 13th Gen back in and made sure it would POST.

## Memory

_Corsair Vengeance LPX 32 GB (2 x 16 GB) DDR4-3600 CL18 Memory ($104.99 @ Amazon)_

2 x 16GB leaves to slots open for future expansion.

After doing some research, it seems to still be the case that DDR5 does not have a significant performance benefit over DDR4 in 'real-world' scenarios. DDR4 is significantly cheaper than DDR5. This particular ram was chosen because it is 'low-profile' which should helps it fit under the CPU Cooler. Also, when comparing the price and speed, this 3600 seems to be the sweet spot. Moving up to 4000 or 4400 adds approximately $80, which is a huge increase.

## Storage

_Western Digital Black SN850 500 GB M.2-2280 PCIe 4.0 X4 NVME Solid State Drive ($69.98 @ Amazon)_

_Western Digital Black SN770 500 GB M.2-2280 PCIe 4.0 X4 NVME Solid State Drive ($49.99 @ Amazon)_

I chose these because they are so cheap compared to other PCIe 4.0 x4 SSDs. The SN850 is supposed to have slightly better performance for a slightly higher cost when compared to the SN770. The 850 is used for my Linux OS.

This SN770 is slightly cheaper and slightly less performant than the SN850. I use this as my Windows drive. The main reason I am getting one 850 and one 770 is because I want to be able to tell the drives apart when looking at them in the BIOS or other places that only output the model number. By using the 850 for Linux and 770 for Windows I can easily identify them.

500GB seems to be the sweet spot price-wise for these. Going down to 250GB only reduces the price by ~$10 and going up to 1TB doubles the price.

## Graphics Card

I put a [Nvidia 4070 Super from ASUS](https://www.amazon.com/ASUS-DisplayPort-Axial-tech-Auto-Extreme-Technology/dp/B0CQPYSF1V?th=1) in this build. I upgrade GPUs more frequently than the rest of the system. When I build a new system I repupose my existing GPUs that's why this is not in the PC Part Picker or included in the total price.

## Case

_Fractal Design Meshify 2 ATX Mid Tower Case ($165.98 @ Newegg)_

I used a Fractal case in my last build and really liked it. I especially like that this case comes with dust filters on all the intakes. I'm sure I could find a cheaper case with all the same features, but I am paying a couple extra bucks because I just this one looks cool.

## Power Supply

_Corsair RM650 650 W 80+ Gold Certified Fully Modular ATX Power Supply ($118.58 @ Amazon)_

This power supply was chosen because it is the cheapest Gold certified modular power supply. It used to be true (probably still is) that PSUs are much more efficient at lower % utilization. Eventhough PC Part Picker is only estimating 392W usage, I am getting a 650W power supply with the hopes that operating a around 50% load will have good efficiency. Also, I may want to add a low-to-mid tier graphic card in the future.

## Wireless Network Adapter

_Asus PCE-AX3000 802.11a/b/g/n/ac/ax PCIe x1 Wi-Fi Adapter ($39.99 @ Amazon)_

This Wireless Network Adapter was chosen because several people in the Amazon reviews said it works well on Linux (kernel >= 5.1).
