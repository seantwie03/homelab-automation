### Motherboard

~_Asus Z790 GAMING WIFI7 ATX LGA1700 Motherboard ($179.99 @ NewEgg)_~

~This motherboard was chosen because it supports 14th Gen Intel without a BIOS update. It is also very similar to the motherboard I used in the desktop22 build.~

This motherboard was a bad choice. It only includes one monitor output (maybe two if using USB-C, but I do not want to burn the one USB-C port on a monitor). Also, I had to return this motherboard. The LED kept flashing. I tested the RAM and CPU, they both passed. I installed Windows on the boot device and that worked OK. The PSU is from a previous build and has been working well for years. I think it has to be the mobo.

_GIGABYTE Z790 Eagle AX LGA 1700 Intel Z790 ($169.99 @ Amazon)_

This board also support 14th Gen Intel. It has two display out. It has [Q-Flash Plus](https://www.youtube.com/watch?v=hDQHeOWZQzE) to flash the bios w/o needing to boot.
