# Device Naming

For the most part I name my devices like this: {MODEL}{YEAR}

{MODEL} is the model of device

{YEAR} is the year it was released

I've never purchased two of the same device before. If I ever do that I will probably regret these decisions.

## Examples

- X1 Carbon 6th Gen came out in 2018 so the hostname is `x1carbon2018`
- I built a custom desktop in 2022 so the hostname is `desktop22`

## Caveat

For my ODroid H3+ I named it `odroidh3plus`. I guess it technically should be called `odroid22` (model and year of release) to be more consistent.. or is `h3` the model? If `h3` is the model, well all h3s came out in 2022, so calling it h3plus22 is a bit redundant.
