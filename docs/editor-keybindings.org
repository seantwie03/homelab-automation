current buffer. In Neovim, it is the nearest ancestor of the current file that
contains a =.git= directory or =.project= marker.

* Legend

- *Key*: Key combination used in both Emacs and Neovim.
- *Mode*: The single Neovim mode in which the row applies. Bindings shared by multiple modes have one row per mode.
- *Emacs Provider*: Package that provides the Emacs function. =builtin= means the function ships with Emacs, and =custom= means it is defined in this configuration.
- *Emacs Function*: Emacs function or prefix map invoked by the key.
- *Nvim Provider*: Package that provides the Neovim function or command. =builtin= means it ships with Neovim, and =custom= means it is defined in this configuration.
- *Nvim Function*: Neovim function invoked by the key. For mappings composed directly from Normal-mode commands or Ex commands, this column shows the mapping's command sequence.
- Key values and function or command names are intentionally written as plain text instead of Org verbatim text to reduce visual noise in the table.
- =N/I=: Not implemented in this editor yet.
- =N/A=: Not applicable to this editor.
